# Inspire Translations

A professional language services website for **Inspire Translations** — Tanzania's premier translation and interpretation company.

**Domain:** [inspiretranslations.co.tz](https://inspiretranslations.co.tz)

## Tech Stack

- **Framework:** React 18 + Vite
- **UI Library:** MUI (Material UI v5)
- **Animations:** Framer Motion
- **Routing:** React Router v6
- **SEO:** react-helmet-async

## Features

- 🌍 Full SEO with JSON-LD structured data, Open Graph, sitemap
- ✨ Lionbridge-inspired animated hero section (parallax card columns + light streaks)
- 🎯 Lottie animated icons throughout
- 💀 Skeleton loading + lazy loading on all pages
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Route-level code splitting

## Pages

Home · About · Services · Projects · Gallery · Blog · Contact · Request a Quote · Privacy Policy · Terms · 404

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```
